Cspray\Labrador\Plugin\Pluggable::hasPluginBeenRegistered

Determine if a Plugin has been registered or not.

Please note that this MAY NOT return only values that have been passed to registerPlugin. All dependencies of PluginDependentPlugins MUST BE implicitly registered as part of their loading process.
